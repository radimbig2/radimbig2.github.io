# Pinchana Twitter

Pinchana Twitter is an unofficial fork prepared so the original Pinchana Twitter/X scraper plugin can be installed as a PyPI package.

My work here was focused on [[stack/python|Python]] packaging, exposing the plugin as an importable dependency, and keeping it usable in downstream projects. This page does not claim authorship of the original scraper logic.

## Stack

- [[stack/python|Python]]
- FastAPI-style plugin API
- X/Twitter GraphQL-related scraping flow
- Gluetun-related resilience
- [[stack/devops|PyPI packaging workflow]]

## Highlights

- PyPI-ready package distribution for an upstream plugin.
- Importable Python wrapper for downstream use.
- Packaging metadata and release workflow setup.
- Clear attribution boundary around the original scraper logic.
