# RadymDownloadBot

RadymDownloadBot is a [[stack/python|Python]] Telegram bot that downloads media from TikTok, Instagram, YouTube, and X/Twitter links. It uses whitelist-based access control so only authorized users can use the bot.

## Stack

- [[stack/python|Python]]
- Telegram Bot API
- yt-dlp
- FFmpeg
- pytest
- Environment-based configuration

## Highlights

- Media downloads from major social platforms.
- Whitelist and admin command system.
- `.env` configuration for deployment.
- Tests for core command and downloader behavior.
