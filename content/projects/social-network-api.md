# Social Network API

Social Network API is a [[stack/csharp|C#]] backend project built with .NET Core and Entity Framework. It provides endpoints for user management, authentication, subscriptions, static files, avatar uploads, and API testing through Postman.

## Stack

- [[stack/csharp|C#]]
- [[stack/backend|ASP.NET Core Web API]]
- Entity Framework
- MySQL
- MediatR
- FluentValidation
- JWT authentication
- Magick.NET

## Why It Matters

This project shows backend API design, authentication, database integration, validation, file handling, and structured request workflows.
