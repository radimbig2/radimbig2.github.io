# BigFitness

BigFitness is a mobile health tracking application built with [[stack/csharp|.NET MAUI and Blazor Hybrid]]. It helps users track daily calories, macronutrients, food entries, body weight history, and personal calorie goals.

## Stack

- [[stack/csharp|C#]]
- .NET 9
- .NET MAUI
- Blazor Hybrid
- SQLite
- Entity Framework Core
- Blazor-ApexCharts

## Highlights

- Daily calorie and macronutrient tracking.
- Product database and food journal.
- Weight history charts.
- Android and Windows support.
