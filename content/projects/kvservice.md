# KVService

KVService is a private full-stack system for managing apartment cleaning operations. It provides secure workflows for managers, employees, and administrators.

## Stack

- [[stack/csharp|C#]]
- [[stack/backend|ASP.NET Core]]
- PostgreSQL
- Entity Framework Core
- Microsoft Identity
- JWT
- DDD-style architecture

## Highlights

- Apartment assignment and deadline management.
- Cleaning status tracking with text or photo evidence.
- Role-based workflows for administrators, managers, and workers.
- Layered Domain, Application, Infrastructure, and API structure.
