# ScreenshotMaker

ScreenshotMaker is a Windows desktop utility for creating screenshots with hotkey support.

## Stack

- [[stack/csharp|C#]]
- Windows Forms
- .NET Framework 4.7.2
- Screna
- MouseKeyHook / hotkey libraries

## Highlights

- Classic Windows desktop application.
- Screen capture workflow.
- Keyboard hook and hotkey support.
