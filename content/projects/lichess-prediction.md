# Lichess Prediction

Lichess Prediction is a [[stack/csharp|C#]] tool that connects to the Lichess API and uses a local Stockfish engine to provide chess position predictions.

## Stack

- [[stack/csharp|C#]]
- Lichess API
- Stockfish
- Local configuration files

## Highlights

- Uses a Lichess API token with read permissions.
- Integrates a local Stockfish executable.
- Provides prediction output for active chess games.
