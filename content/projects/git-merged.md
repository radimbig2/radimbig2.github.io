# git-merged

`git-merged` is a command-line utility that finds local or remote Git branches already merged into a target branch. It helps developers identify branches that are safe to clean up.

## Stack

- Shell
- PowerShell
- npm package distribution
- [[stack/devops|Git workflow automation]]

## Highlights

- Detects branches merged into `main`, `develop`, `production`, or a custom branch.
- Supports local and remote branch scanning.
- Can save reports or print directly.
- Optional deletion flow with confirmation and author filtering.
