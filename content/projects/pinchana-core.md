# Pinchana Core

Pinchana Core is an unofficial fork prepared so the original Pinchana Core project can be installed as a PyPI package.

My work here was focused on [[stack/python|Python]] packaging, distribution metadata, dependency setup, and making the package usable as a dependency for downstream projects. This page does not claim authorship of the original Pinchana Core project.

## Stack

- [[stack/python|Python]]
- Pydantic
- Docker-related tooling
- Gluetun-related integration
- [[stack/devops|PyPI packaging workflow]]

## Highlights

- PyPI-ready distribution for an upstream project.
- Preserves the original `pinchana_core` import path.
- Packaging metadata and dependency setup.
- Intended for Pinchana-based downstream projects.
