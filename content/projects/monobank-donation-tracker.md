# Monobank Donation Tracker

Monobank Donation Tracker is a streamer-focused [[stack/python|Python]] application that monitors Monobank donations in real time, displays donation alerts in OBS overlays, and can play YouTube music links submitted through donation comments.

## Stack

- [[stack/python|Python]]
- aiohttp
- PyQt5
- HTML / CSS / JavaScript
- OBS browser overlays

## Highlights

- Real-time donation notifications.
- OBS overlay and live donation feed pages.
- YouTube music queue with caching and playback controls.
- CLI and GUI control interfaces.
