# TimeManager

TimeManager is a Windows desktop time management application built as a layered [[stack/csharp|C#]] solution.

## Stack

- [[stack/csharp|C#]]
- WPF
- Multi-project solution architecture
- Database layer

## Structure

- `TimeManager.Core` - core application logic.
- `TimeManager.Database` - persistence and database layer.
- `TimeManager.WPF` - desktop user interface.
