# AI-Assisted Workflow

I use AI tools as practical engineering assistants, not just as generic chatbots. My workflow includes [[stack/ai-tools|Codex, Claude, Cursor, MCP servers, and custom skills]].

## How I Use AI

- Navigate unfamiliar codebases faster.
- Trace bugs through controllers, services, mappers, database queries, and UI flows.
- Generate implementation plans and compare tradeoffs before coding.
- Refactor code while preserving existing project style.
- Draft documentation, READMEs, API notes, and portfolio content.
- Use project-specific MCP servers and skills to connect AI assistants to local tools, databases, repositories, and repeatable workflows.

## Result

AI helps me shorten feedback loops: faster analysis, cleaner implementation planning, better documentation, and more reliable verification.
